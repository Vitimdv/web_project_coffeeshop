# Triple Espresso

## 📖 Descrição

Projeto de uma página web de cafeteria desenvolvido com HTML e CSS, com foco em estrutura semântica e metodologia BEM.

## Tecnologias

- HTML5
- CSS3
- BEM
- Normalize.css

## Funcionalidades

- Layout organizado
- Seções bem definidas
- Formulário de reserva
- Links de redes sociais

## Melhorias futuras

- Tornar responsivo
- Adicionar JavaScript
- Melhorar acessibilidade
